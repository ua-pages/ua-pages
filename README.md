## Напрями

- **Developer Tooling** — CLI, локальні менеджери й інструменти для роботи з проєктами.
- **Platform Engineering** — основа, на якій команда може будувати й розвивати продукт.
- **Вебсистеми** — підтримувані застосунки з прозорою архітектурою та передбачуваним станом.
- **Backend та API** — прості Node.js API вже створюю; складніші серверні системи залишаються напрямом розвитку.

## Принципи

- Кожна залежність — архітектурне рішення, а не автоматичний вибір.
- AI — робочий інструмент, а не професійна ідентичність.
- Підтримуваність важливіша за короткочасне враження.
- Local-first доречний там, де він справді спрощує систему.
- Українська може бути природною мовою інженерного мислення.

ua-pages — не агентство й не каталог технологій. Це моя інженерна екосистема: робочі інструменти, відкриті проєкти та підходи, перевірені практикою.

Я допомагаю створювати рішення, які:

# Principles

## Local First

Продукти повинні працювати локально та зберігати дані у користувача.

* Дані належать користувачу
* Без обов'язкових акаунтів
* Без прив'язки до хмарних сервісів
* Простий експорт та імпорт даних

## Lightweight First

Продукти повинні залишатися простими та легкими.

* Мінімум залежностей
* Мінімум складності
* Прозора архітектура
* Легка підтримка

## Fast First

Продукти повинні бути швидкими.

* Швидкий запуск
* Швидкий інтерфейс
* Швидка розробка
* Швидке розуміння кодової бази

## Focused First

Інструменти створюються розробником для розробників.

* Одна задача
* Чіткі межі
* Без feature creep

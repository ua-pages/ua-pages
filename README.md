## ПРОФІЛЬ

Досвідчений універсальний програміст з ухилом на клієнтьску частину та багаторічним досвідом у веб-розробці. Сильний досвід у побудові масштабованих веб-застосунків, архітектурі односторінкових та серверно-рендерених рішень, оптимізації продуктивності та технічному лідерстві.

Основна експертиза — Cучасні вебсайти, клієнтські та сереверні частини, робота з базами данних, інтеграція із серверами, мікросерверами. Практичний досвід роботи є і з модульною архітектурою серверів.

Комфортно працюю у продуктових та великих-командах — беру відповідальність за технічні рішення, якість коду та наставництво. Активно використовую ШІ-інструменти для прискорення розробки та підвищення якості.
